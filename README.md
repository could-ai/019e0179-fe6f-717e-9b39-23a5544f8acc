# Lessons App

A simple Flutter application demonstrating how to structure lessons with indexed contents.

## Features
- List of available lessons/courses.
- Detailed view for each lesson.
- Each content item inside a lesson is cleanly indexed and displayed with its numerical order.

## Tech Stack
- Flutter
- Dart

## Run Instructions
1. Run `flutter pub get`
2. Run `flutter run`
